Visualization

Circle Network

Epistatic effect results can be represented in circle network. One node in circle network represents a feature in epistatic effect results table. All nodes are arranged as a circle. Two nodes that have epistatic relationship are linked by an undirected edge. It is easy to know that the number of edges of one node equals to the number of epsitatic effects that one feature has on other features. Circle network has two main advantages comparing to table:

  1. It is easy to find out which feature has the most epistatic effects on other features.
  2. It is easy to find complex indirected relationship among features.
Mouse over any of nodes in this network to see its associative features in red.

Click one node to find details in above Epistatic Effect table. Double click on graph back to default.

Adjacency Matrix

Epistatic effect results also can be represented by an adjacency matrix, where row and column names are the features in epistatic effect table, each cell (i, j) represents an edge from vertex i to vertex j. Matrix cell depicts coefficent value in epistatic effect table. The darker cells indicate strong interaction. While circle network can give a good view for node-link relationship, adjacency matrix has other advantages:

  1. Adjacency matrix can give a whole picture of epistatic relationship, especially for large number of epistatic features. As circle network gets large and highly connected, it often becomes into giant hairballs of line crossings.
  2. Matrix cell value can represent the strength of epistatic relationship. And it also can be encoded to show additional data.


Use the drop-down menu to reorder the matrix.

Force-directed Graph

Force-directed graph is a more flexible way to represent epistatic relationship. One node in graph represents a feature in epistatic effect table. Two nodes that have epistatic relationship are linked by an edge. It is to position the nodes of a graph in two-dimensional space so that all the edges are of equal length and there are as few crossing edges as possible, by assigning forces among the set of edges and nodes, based on their relative positions.
