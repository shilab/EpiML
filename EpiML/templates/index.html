An open machine learning web service for pairwise epistatic analysis

EpiML can detect main effect and pairwise epistatic effect by using several machine learning and statistical methods and provide many interactive visualizations to help analyze epistatic relationship. Especially for genotype and microRNA case studies, external resources are integrated for result analysis.

Easy to Use

Describe job and upload data. Prepare data in requirement format. All user's data will be kept private.

Select a machine learning method. Several popular machine learning and statistical methods are built-in.

Automatically process in background. You don't need to care about the details. Our server will handle all the processing in background.

Check out results.
You can directly analyze results using interactive visualization on result page or download PDF report.

Interactive Visualization

Visualizations are very useful tools to help analyze epistatic relationship. EpiML provides friendly interactive visualizations, like circle network, adjacent matrix and force-directed graph. When you move mouse over one node, more detailed information and linked nodes will be high light.

Multiple Resources Integration

Multiple external resources are integrated. UCSC genome browser links are provided for genetic data. miRBase and miR2Disease database are embedded for microRNA data. Multiple resources integration will significantly reduce user time for multi-step analysis.

Customize Your Analysis

For each job, its corresponding Jupyter notebook is automatically generated. All notebooks can be downloaded, modified and rerun, allowing users to fully customize their epistasis analysis on local computers. To achieve this, we provide a pre-configured Docker container containing a fully functional Jupyter server with all the Python and R libraries necessary to run the notebooks.
