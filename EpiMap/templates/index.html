Epistatic Analysis Sever: An open machine learning web service and model repository for the epistatic analysis

This server provides epistatic analysis, through which biologists and biomedical researchers can train a specific model based on their own data or predict their new data based on a shared model in model repository, and visualize results using interactive visualizations.

Model Repository

Trained models are stored, and they are convenient to share with others. Due to the challenges of replicating large models on large datasets, it is hard to achieve a good model following existing work. We will provide a model repository which is a set of unified containers with trained models. We will package the depending packages, configurations, model parameters, and training details in the container, which can be run on any operating system.

Interactive Visualization

Results are visualized. To help understand the results, we will provide interactive visualization of intermediate results and final results, with options to zoom in and out for diverse views of the data and results.

Multiple Resources Integration

Multiple external resources are integrated for analysis. Many external resources are embedded in results visualization. We also provide links to metadata of omics data and external resources, such as gene annotations and biological pathways, to help interpret the results.
