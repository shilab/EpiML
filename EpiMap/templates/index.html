Epistasis Analysis Server: An open machine learning web service and model repository for epistasis analysis

Epistasis analysis is a foundation in the analysis of genetic networks. The term “epistasis” is sometimes used to describe some form of statistical interaction between genetic factors and is alternatively sometimes used to describe instances in which the effect of a particular genetic variant is masked by a variant at another locus. Here, we define epistasis as the situations that the phenotype variance could be explained by the interactions or combinations of (epi-)genetic variants, instead of individual (epi-)genetic variants alone.

This server provides pair-wise epistasis analysis. First, we identify the main effects using a model with only main effect on the phenotype. Second, a corrected phenotype is calculated by removing the significant main effect from the original phenotype. Third, we select features with epistatic effect on the corrected phenotype. Finally, we run the full model with main and epistatic effects on the previously selected main and epistatic features.

Model Repository

Trained models are stored, and they are convenient to share with others. Due to the challenges of replicating large models on large datasets, it is hard to achieve a good model following existing work. We will provide a model repository which is a set of unified containers with trained models. We will package the depending packages, configurations, model parameters, and training details in the container, which can be run on any operating system.

Interactive Visualization

Results are visualized. To help understand the results, we will provide interactive visualization of intermediate results and final results, with options to zoom in and out for diverse views of the data and results.

Multiple Resources Integration

Multiple external resources are integrated for analysis. Many external resources are embedded in results visualization. We also provide links to metadata of omics data and external resources, such as gene annotations and biological pathways, to help interpret the results.
